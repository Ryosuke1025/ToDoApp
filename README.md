## 概要
イベントに日付と曜日を指定してToDoリストのようにWEB上(データベース)に保存できるアプリ

・JDBCドライバ(Javaからデータベースにアクセスできる標準インターフェース)について
・JSP(HTMLのプログラムにJavaのプログラムを埋め込める技術)について
・Apache(Webサーバー)とTomcatの関係
